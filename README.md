# Poker Hand Logger (GUI v11)

シンプルにポーカーのハンド履歴を記録・分析するための、**ブラウザのみで動作するアプリ**です。
スマホ対応・PWA対応（ホーム画面追加）も可能です。

---

## 概要

* フロントのみ（HTML / CSS / JavaScript）
* データはブラウザの **IndexedDB** に保存
* オフライン利用可能（Service Worker）
* GitHub Pages などで公開すればスマホから利用可能

---

## 主な機能

### 1. ハンド記録

* 2枚のカードを選択（1枚ずつ選択式）
* Position / Action / Memo を記録
* Unknownカード対応

---

### 2. 履歴表示

* セッションごとに整理
* 日付 / Position / Action でフィルタ可能
* 各ハンド削除可能

---

### 3. 統計

以下を自動計算：

* **VPIP**
* **PFR**
* **3bet+%**

  * 分子: `3bet + 4bet + All-in`
  * 分母: `Call + 3bet + Call3 + 4bet + Call4 + All-in`

---

### 4. ハンド頻度ヒートマップ（NEW）

13×13のレンジ形式で表示：

* ペア（例: `JJ`）
* スーテッド（例: `A8s`）
* オフスート（例: `K7o`）

特徴：

* 出現回数が多いほど色が濃くなる
* フィルターと連動
* 実戦の配牌傾向が一目で分かる

---

### 5. セッション管理

* 日付 + 任意の名前で管理
* 種別（Tournament / Cash / Study など）
* セッション単位で削除可能

---

### 6. CSV入出力

* エクスポート（分析用）
* インポート（データ復元）

---

### 7. PWA対応

* ホーム画面に追加可能
* オフラインでも動作

---

## 使い方

### 1. 起動

```bash
index.html をブラウザで開く
```

---

### 2. 基本操作

1. セッションを選択 or 作成
2. カードを2枚選択
3. Position / Action を選択
4. 必要ならMemo入力
5. 保存

---

### 3. フィルタ

* 日付
* Position
* Action

---

## スマホで使う方法（推奨）

### GitHub Pages

1. GitHubリポジトリにアップロード
2. `Settings → Pages` を有効化
3. 公開URLにアクセス
4. 「ホーム画面に追加」

---

## データについて

* 保存先: **ブラウザのIndexedDB**
* サーバーには送信されません
* ブラウザを消すと消えます

---

## 注意点

* `file://` で開くとPWA機能は制限されます
* HTTPS環境（GitHub Pagesなど）推奨
* `icon.svg` が必要（PWA用）
