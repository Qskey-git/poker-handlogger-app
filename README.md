# Poker Hand Logger

シンプルにポーカーのハンド履歴を記録・分析するための、**ブラウザのみで動作するアプリ**です。
スマホ対応・PWA対応（ホーム画面追加）も可能です。

---

## 概要

* フロントのみ（HTML / CSS / JavaScript）
* データはブラウザの **IndexedDB** に保存
* オフライン利用可能（Service Worker）

---

## 主な機能

### 1. ハンド記録

* 2枚のカードを選択（1枚ずつ選択式）
* Position / Action / Memo を記録
* Unknownカード対応

---

### 2. 履歴表示

* セッションごとに整理
* Position / Action でフィルタ可能
* 各ハンド編集と削除可能

---

### 3. ハンド頻度ヒートマップ

13×13のレンジ形式で表示：

* ペア（例: `JJ`）
* スーテッド（例: `A8s`）
* オフスート（例: `K7o`）

特徴：

* 出現回数が多いほど色が濃くなる
* フィルターと連動

---

### 4. 統計

以下を自動計算：

* **VPIP**
  * 分子: `Open + Call + 3bet + Call3 + 4bet+ + Call4+ + FoldR`
  * 分母: `Actionが入力されたすべてのハンド`
* **PFR**
  * 分子: `Open + 3bet + 4bet+ + FoldR`
  * 分母: `Actionが入力されたすべてのハンド`
* **3bet+%**

  * 分子: `3bet + 4bet+`
  * 分母: `Call + 3bet + Call3 + 4bet+ + Call4`

---

### 5. セッション管理

* 日付 + Note + Memoで管理
* 種別（TTM / Cash / Other）
* セッション単位で削除可能

---

### 6. CSV入出力

* エクスポート（データ保存）
* インポート（データ復元）

---

### 7. PWA対応

* ホーム画面に追加可能
* オフラインでも動作

---

## 使い方

### 1. 起動

```bash
index.html をブラウザで開く
```

---

### 2. 基本操作

1. セッションを選択 or 作成
2. カードを2枚選択
3. Position / Action を選択
4. 必要ならMemo入力
5. 保存

---

### 3. フィルタ

* Position
* Action

---

## データについて

* 保存先: **ブラウザのIndexedDB**
* サーバーには送信されません
* ブラウザの該当履歴を消すと消えます

---
